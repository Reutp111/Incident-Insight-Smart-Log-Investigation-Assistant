# Incident Insight

Incident Insight is a smart log investigation assistant designed to help support, operations, and technical teams understand logs more easily.

The tool supports:
- Uploading one or multiple log files
- Pasting raw log text directly
- Automatic detection of repeated IPs, repeated indicators, and likely issue categories
- Custom glossary support for organization-specific terms
- Suggested follow-up keywords
- Local investigation assistant for explaining findings, next checks, and case notes
- Copying and downloading investigation summaries

This project was built to make log analysis more accessible, especially for first-time users or support teams who need fast, readable insights instead of raw log lines.
